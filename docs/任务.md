### 5.2 后端接口

| 方法 | 路径 | 请求参数 | 响应 data | 说明 |
|------|------|----------|-----------|------|
| GET | `/user/profile` | Header Token | `{ id, name, phone, email, avatar, gender, birthday }` | 获取个人信息 |
| PUT | `/user/profile` | `{ name, gender, birthday, email, avatar }` | `null` | 修改个人信息 |
| GET | `/family-members` | — | `[{ id, name, phone, relation, gender, birthday, idCard, isDefault }]` | 就诊人列表 |
| POST | `/family-members` | `{ name, phone, relation, gender, birthday, idCard, isDefault }` | `{ id }` | 新增就诊人 |
| PUT | `/family-members/{id}` | `{ name, phone, relation, gender, birthday, idCard, isDefault }` | `null` | 编辑就诊人 |
| DELETE | `/family-members/{id}` | 路径参数 | `null` | 删除就诊人 |
| GET | `/appointments/my` | `page, pageSize, status` | 分页挂号列表 | 我的挂号（调用人员3接口） |
| POST | `/appointments/{orderNo}/cancel` | 路径参数 | `null` | 取消挂号（调用人员3接口） |
| GET | `/consults/my` | `page, pageSize, status` | 分页咨询列表 | 我的咨询（调用人员3接口） |
| GET | `/reviews/my` | `page, pageSize` | 分页 | 我的评价 |
| POST | `/reviews` | `{ orderType, orderId, doctorId, content, rating }` | `{ id }` | 提交评价 |
| GET | `/messages` | `page, pageSize` | 分页 | 消息列表 |
| POST | `/messages/{id}/read` | 路径参数 | `null` | 标记已读 |
| GET | `/messages/unread-count` | — | `{ count }` | 未读消息数（备选） |
| GET | `/feedbacks` | `page, pageSize` | 分页 | 历史反馈 |
| POST | `/feedbacks` | `{ type, content, images }` | `{ id }` | 提交反馈 |
| GET | `/follow/my` | `type(1医院/2医生/3疾病), page, pageSize` | 分页 | 关注列表 |
| POST | `/follow/{type}/{id}` | 路径参数 | `{ id }` | 添加关注 |
| DELETE | `/follow/{type}/{id}` | 路径参数 | `null` | 取消关注 |

### 5.3 数据库表
`t_user` · `t_family_member` · `t_follow` · `t_review` · `t_feedback` · `t_message`

### 5.4 改造要点（来自代码审计）

| 问题 | 改造方案 |
|------|----------|
| `personal-info.html` 个人信息写死在页面中 | 改为 GET `/user/profile` 加载，修改后 PUT |
| `family-members.html` 就诊人写死 3 条 | 替换为 GET `/family-members`，CRUD 全走 API |
| `my-appointment.html` 订单列表 Mock | 替换为 GET `/appointments/my`，取消订单 POST `/appointments/{orderNo}/cancel` |
| `my-reviews.html` 评价数据硬编码 | 替换为 GET `/reviews/my`，写评价 POST `/reviews` |
| 关注页数据全部写死 | 替换为 GET `/follow/my?type=1/2/3`，取消关注 DELETE `/follow/{type}/{id}` |
| `my-messages.html` 消息列表 Mock | 替换为 GET `/messages`，点击标记已读 |
| `my-feedback.html` 反馈历史 Mock | 替换为 GET `/feedbacks`，提交 POST `/feedbacks`（支持图片列表） |
| Sidebar 无高亮逻辑 | 根据当前 URL 匹配菜单，加高亮 class |
| 所有页面均未校验登录态 | 抽到公共鉴权脚本统一调用 `/auth/me`；受限页面初始化时校验，无 Token 直接跳转登录页 |
